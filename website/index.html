Workflow

Four Steps to Perfect Sync

1

Import

Create tracks for each recording device and add your audio or video files. Drag-and-drop or use the file picker. Video audio is extracted automatically. Supports WAV, AIFF, FLAC, MP3, MP4, MOV, MKV, and many more formats.

2

Analyze

One click runs FFT cross-correlation at 8 kHz ‚Äî 36x faster than full resolution while maintaining sub-millisecond precision. Watch live progress with per-clip confidence scores and ETA.

3

Sync

Each track's clips are stitched into a single continuous audio stream at full resolution. Silence fills the gaps between clips, and crossfades smooth the transitions.

4

Export

Export one synced audio file per track ‚Äî all the same length and perfectly aligned. Choose WAV, AIFF, or FLAC with 16/24/32-bit depth. Drop the files into any DAW or video editor.
